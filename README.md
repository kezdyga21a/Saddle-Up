Fontos: utasítások a játék elindításához

1: Legyen letöltve a Python, és a Microsoft Visual Studio

2: A letöltött .zip fájlt bontsd ki

3: Nyisd meg a Visual Studiot

4: A kibontott mappából húzd be a Visual Studioba a main.py és fighter.py fájlokat

5: A Visual Studioban menj a main.py nevű oldalra

6: A felső csíkból válaszd ki a run, majd a start debugging lehetőséget

7: Ha mindent jól csináltál, jobboldalt megjelent egy háromszög alakú ikon. Nyomjál rá, és élvezd a Saadle Up!-ot
